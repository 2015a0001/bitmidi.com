### Live site: [https://nodefoo.com](https://nodefoo.com)

## Contributing

It's easy to run the code!

```bash
git clone git@github.com:feross/nodefoo.com.git
cd nodefoo.com/
npm install
```

Run the server:

```bash
npm run build
npm start
```

Watch and restart automatically:

```bash
npm run watch
```
